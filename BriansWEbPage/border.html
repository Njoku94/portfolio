This page is about how
CSS Border

Bordering in CSS allows a Developer to specify the style, width, & color of an element's bordering.
Border-style specifies what kind of border to display which comes in four value :

  1. Top Border

  2. Right Border

  3. Bottom Border

  4. Left Border
