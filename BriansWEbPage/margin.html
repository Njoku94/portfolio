This page is about how
CSS Padding
works.

Margins in CSS allows a Developer to create space around elements, that are outside of any defined border, because of this you full control over the pages margins. Thus, all the margin properties can have the following values:

  • auto allow the browser calculates the margin
  • length allow one to specify the margins diameter
  • % allows one to specify the margin width by %
  • inherit allows the margin to be inherited by its parent element
