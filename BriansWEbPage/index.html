Content

Content within the CSS Box Model is simple; it's everything that's in the center of everything else. For example, the text on this page would be the content.
It can be simply defined as the innermost elements of the box. This can refer to images, text, and anything else that is not in other parts of the box.
We can look at it like this:
I'm the Content!
I'm some padding!
I'm the border!
And I'm the margin!
Where we only care about the content for now.
Even the text you're currently reading is content! It has other things that can be added, like padding, but for now it's just simple text.
We also need to know what is not content. Actual padding is not content, nor are borders or things like spacing and outlines. Those are done in other parts of the box.
