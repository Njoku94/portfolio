This page is about how
CSS Padding
works.

Padding in CSS allows a Developer to create space around an element's content, that are inside of any defined border, because of this you full control over the pages padding. Thus, all the padding properties can have the following values:

  • length allow one to specify the margins diameter
  • % allows one to specify the margin width by %
  • inherit allows the margin to be inherited by its parent element
Negative values are not allowed!!!
